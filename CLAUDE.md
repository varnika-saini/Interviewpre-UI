# CLAUDE.md

## Project Name

InterviewPrep Pro

## Project Goal

Build a modern, professional, and responsive Interview Preparation Platform UI using Next.js 15, TypeScript, Tailwind CSS, and Shadcn UI.

The project should focus entirely on frontend development with realistic mock data.

No backend implementation is required.

---

## Tech Stack

* Next.js 15 (App Router)
* TypeScript
* Tailwind CSS
* Shadcn UI
* Framer Motion
* React Query (mock data)
* Zustand
* Recharts
